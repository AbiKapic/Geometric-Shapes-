# Geometric Shapes
## Description
Geometric Shapes in Java using inheritance and exceptions

## Content
- Java OOP and Inheritance
- Exception classes
- Easily to import packages
- Override
- Try and Catch


## Usage
- Clone repository
- Use it in any available java compiler it should work easily
